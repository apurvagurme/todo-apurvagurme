Delete Todo

Click on the trash "ðŸ—‘" icon right to the title of a todo to initiate deletion process of a todo. Then it will ask you for confirmation of deletion. Then click on "Ok" button to permanently delete the todo. Or click on "Cancel" button to cancel the deletion process.

Add Task

To add task, click on "New Task" bar at the bottom of any todo and it will let you type a name for the task. after typing a name for the task, press enter and you will have the task under the todo.

Rename Task

Renaming task is as same as renaming todo. Just go over a task name, click on it, modify and click outside the task name or press 'Enter' key.

Mark Task as done/undone

You can mark a task as done and undo it to undone. To do so, use the little checkbox left to any task name. If a task is done, it's checkbox will be checked and there will be line through over the task name and it will be faded. Thus you can differentiate between done and not-done tasks.

Search Task/Todo

Searching Todo by title or Task by task name is so simple. At top-right corner of the screen you'll notice the hybrid search bar that can be used for both Todo and Task search. A toggle button, right to the search bar is used switch between Todo Search Mode and Task Search Mode. Default text on the search bar will show you which mode you are in.
